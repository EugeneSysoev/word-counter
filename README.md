# Word Counter
Консольная утилита для подсчета слов в текстовых файлах.

# Функциональность
Подсчет общего количества слов в txt-файлах
Игнорирование знаков препинания
Исключение союзов и частицы "не"
Слова с дефисами считаются как одно слово
Вывод топ-5 самых частых слов по флагу -t

# Скопируй репозиторий с GitHub
git clone https://github.com/EugeneSysoev/word-counter.git

# Перейди в папку проекта
cd word-counter

# Установи зависимости
npm install

# Собери проект
npm run build

# Использование через npm скрипты
npm start -- test.txt

npm start -- test.txt -t

# Примеры работы
Общее количество слов: 151

Топ-5 самых частых слов:
1. "обеспечивает" - 4 повтор.
2. "развития" - 4 повтор.
3. "деятельности" - 3 повтор.
4. "формировании" - 3 повтор.
5. "нашей" - 3 повтор.
